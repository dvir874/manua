# KORE — Booking Calendar Flow
## יומן פגישות אוטומטי — הגדרה מלאה

---

## הרעיון

ליד נכנס → מקבל הודעת WhatsApp → לוחץ לינק → בוחר זמן → נקבעת פגישה ביומן שלך אוטומטית.
אפס back-and-forth. אפס "מתי נוח לך?".

---

## הכלי המומלץ: Cal.com

**למה Cal.com ולא Calendly:**
- חינמי לחלוטין לשימוש בסיסי
- ממשק בעברית (חלקי)
- API פתוח לחיבור עם n8n / Green API
- אפשרות self-host

---

## הגדרת Cal.com — שלב אחר שלב

### שלב 1: יצירת חשבון
1. כנס ל-[cal.com](https://cal.com) → Sign Up
2. הגדר שם משתמש: `[שם-העסק]` → יהיה ה-URL: `cal.com/[שם]`
3. חבר את Google Calendar / Outlook שלך

### שלב 2: הגדרת סוג פגישה
1. "New Event Type" → "Discovery Call" / "ייעוץ ראשוני"
2. **משך:** 20 דקות (כיכניס לידים, לא קונסולטציה מלאה)
3. **שעות זמינות:** 09:00–19:00, ראשון–חמישי
4. **אזור זמן:** Israel (Jerusalem)
5. **Buffer:** 15 דקות אחרי כל פגישה (לא יקבעו ישר אחת אחרי השנייה)

### שלב 3: הודעות אוטומטיות (Cal.com שולח מעצמו)
- **אישור:** "הפגישה נקבעה! ב-[תאריך] ב-[שעה]. [לינק zoom/טלפון]"
- **תזכורת 24 שעות לפני:** "מחר ב-[שעה] — נתראה!"
- **תזכורת שעה לפני:** "עוד שעה — מוכן?"

### שלב 4: קישור הזמנה
```
https://cal.com/[שם-העסק]/discovery
```
זה הלינק שמכניסים להודעות WhatsApp.

## Flow ב-Green API / n8n

### Flow 1: ליד חדש → קביעת פגישה

```
ליד חדש נכנס (WhatsApp / טופס)
          ↓
מענה אוטומטי מיידי (תוך 90 שניות):
[הודעת קבלת פנים + לינק Cal.com]
          ↓
לא קבע תוך 24 שעות?
          ↓
תזכורת D+1
          ↓
לא קבע תוך 48 שעות נוספות?
          ↓
תזכורת D+3 (אחרונה)
```

## חיבור Cal.com ל-n8n

```
Cal.com → Webhooks → New Webhook
URL: https://[your-n8n]/webhook/cal-booking
Events: BOOKING_CREATED, BOOKING_CANCELLED
```

### n8n Flow לאחר קביעת פגישה:
1. Webhook מ-Cal.com "BOOKING_CREATED"
2. חלץ: שם, טלפון, תאריך, שעה
3. הוסף ל-CRM → שנה סטטוס ל-"Discovery Scheduled"
4. שלח הודעת אישור ב-WhatsApp (הודעה למעלה)
5. הוסף תזכורת: D-1 ו-D-0 (שעה לפני)

---

## חיבור ל-Google Meet אוטומטי

ב-Cal.com → Event Type → Location → "Google Meet"
כל פגישה תיצור לינק Meet אוטומטי ותשלח בהזמנה.
