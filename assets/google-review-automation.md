# KORE — Google Review Automation
## אוטומציית ביקורות גוגל — הגדרה + הודעות מוכנות

---

## למה זה קריטי

ביקורת גוגל אחת = ליד אורגני חינמי שמגיע לבד.
לקוח שמרוצה — לא תמיד ייזום. אבל אם תשאל בזמן הנכון, ב-80% מהמקרים הוא יכתוב.
**הזמן הנכון:** תוך 24–48 שעות מסיום השירות.

---

## ה-Flow האוטומטי

```
עסקה סגורה / שירות הסתיים
          ↓
  המתן 24 שעות
          ↓
  [הודעה 1 — בקשת ביקורת]
          ↓
  לא ענה? D+3 → [הודעה 2 — תזכורת]
          ↓
  לא ענה? D+7 → [הודעה 3 — last touch]
          ↓
  ענה? → תודה + הפנייה לחבר
```

---

## איך מגדירים ב-Green API / n8n

1. **טריגר:** CRM מסמן עסקה כ-"סגורה" / "הושלם"
2. **המתן 24 שעות:** delay node
3. **שלח הודעה 1** לטלפון הלקוח
4. **בדוק תגובה:** אם לא ענה תוך 48 שעות → שלח הודעה 2
5. **אם לא ענה עוד 72 שעות** → שלח הודעה 3 → סגור את ה-flow

---

## הודעות מוכנות — 3 גרסאות

> **הכנה:** השג לפני-כן את ה-URL של דף גוגל ביזנס של הלקוח.
> `https://g.page/r/[PLACE_ID]/review`
> מוצאים ב: Google Business Profile → "קבל קישור לביקורת"

---

### הודעה 1 — D+1 (24 שעות לאחר הסתיים)

```
היי [שם הלקוח],

תודה על האמון — שמחתי לעזור 🙏

בקשה קטנה: אם היית מרוצה מהשירות, ביקורת קצרה בגוגל עוזרת לי מאוד להגיע ללקוחות נוספים.

לוקח פחות מדקה:
👉 [GOOGLE_REVIEW_LINK]

תודה מראש!
[שם בעל העסק]
```

---

### הודעה 2 — D+3 (תזכורת)

```
[שם], היי שוב 😊

רק תזכורת קצרה על הביקורת בגוגל — אם יש לך דקה:
👉 [GOOGLE_REVIEW_LINK]

לא חייב לכתוב הרבה — גם "שירות מצוין, ממליץ" עוזר מאוד.

תודה!
```

---

### הודעה 3 — D+7 (Last Touch)

```
[שם], הודעה אחרונה ממני בנושא 🙏

אם השירות היה טוב — כל מילה בגוגל שווה לי הרבה:
👉 [GOOGLE_REVIEW_LINK]

ואם משהו לא היה כמו שציפית — אשמח לשמוע ישירות, כדי שאשתפר.

תודה על הזמן!
```

## הודעת תודה — לאחר שכתב ביקורת

> שלח ידנית תוך שעה מקריאת הביקורת

```
[שם]! ראיתי את הביקורת — תודה מהלב 🙏

זה אומר לי המון.

אם אי פעם תצטרך עזרה שוב — אני כאן.
ואם יש מכר שיכול להיעזר בי — אשמח אם תפנה 😊
```

---

## הגדרת הקישור בגוגל ביזנס

**שלב 1:** כנס ל-[Google Business Profile](https://business.google.com)
**שלב 2:** בחר את העסק → "קבל קישור לביקורות"
**שלב 3:** העתק את הקישור → שים ב-`[GOOGLE_REVIEW_LINK]` בכל ההודעות
**שלב 4:** שמור ב-CRM בפרופיל הלקוח

## אוטומציה ב-n8n (Flow מלא)

```json
{
  "trigger": "crm_deal_closed",
  "steps": [
    { "type": "wait", "hours": 24 },
    { "type": "whatsapp_send", "template": "review_1" },
    { "type": "wait", "hours": 48 },
    { "type": "if_no_reply", "then": [
      { "type": "whatsapp_send", "template": "review_2" },
      { "type": "wait", "hours": 72 },
      { "type": "if_no_reply", "then": [
        { "type": "whatsapp_send", "template": "review_3" }
      ]}
    ]}
  ]
}
```
